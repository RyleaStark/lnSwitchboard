Serve LNURL from your domain

Forward visitors from every domain you want to offer Lightning addresses on to the lnSwitchboard endpoint. The admin panel stays local on port 22121.

Copy the /.well-known/lnurlp/ path to each domain and point it at this service. 

server {
    listen 443 ssl;
    server_name your.domain;

    location ~ ^/\.well-known/lnurlp/[a-z0-9._+-]+$ {
        proxy_pass http://127.0.0.1:22121$request_uri;
        proxy_set_header Host $host;
        proxy_set_header X-Forwarded-For $remote_addr;
    }
}
